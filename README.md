<section class="section">
<h2>Contexto do Problema</h2>
<p>Mesmo sendo o maior exportador de jogadores do futebol mundial, o processo de scouting no Brasil ainda depende fortemente de redes informais de contato, observação manual e ferramentas fragmentadas. Scouts frequentemente utilizam combinações de WhatsApp, planilhas e plataformas distintas para reunir dados, vídeos e avaliações técnicas.</p>
<p>Hipótese de Produto: existe uma oportunidade clara para plataformas que integrem vídeo, estatística e relatórios em uma única interface acessível para scouts independentes, clubes menores e academias de formação.</p>
</section>
